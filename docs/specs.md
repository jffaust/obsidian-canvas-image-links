The goal for this plugin is to support links on image files inside the Obsidian Canvas.

This could be a link to a web page, a link to a note/canvas in the vault or even to a file on the computer.

You can find the specification for the Obsidian Canvas JSON format here: https://jsoncanvas.org/spec/1.0/

You can find the developer docs for making Obsidian plugins here: https://docs.obsidian.md/Plugins/Getting+started/Build+a+plugin
